---
title: Test a Source
api: "GET https://missourimonster-vyla-api.hf.space/api/test/{id}"
description: Debug a single provider in isolation — get the raw URL, proxy URL, elapsed time, and error details.
---

Tests one specific provider for a given TMDB ID without running the full parallel fanout or live verification. Useful for debugging why a source isn't appearing in movie or TV responses, or for checking provider availability before building.

<Note>
  This endpoint does **not** verify the returned URL. Unlike `/api/movie` and `/api/tv`, it returns whatever the provider gives back — including potentially broken URLs. Use `ok: true` as a soft indicator only.
</Note>

---

## Path Parameters

<ParamField path="id" type="string" required>
  TMDB ID. For movies, use the movie ID. For TV, use the series ID.
</ParamField>

---

## Query Parameters

<ParamField query="source" type="string" required>
  Provider key to test. Must match one of the configured source keys exactly.

  Available values: `source1`, `source2`, `source3`, `source4`, `source5`, `source6`
</ParamField>

<ParamField query="season" type="number">
  Season number. Required for TV episodes; omit for movies.
</ParamField>

<ParamField query="episode" type="number">
  Episode number. Required for TV episodes; omit for movies.
</ParamField>

## Request

<CodeGroup>
```bash Movie
curl "https://missourimonster-vyla-api.hf.space/api/test/550?source=source1"
```

```bash TV Episode
curl "https://missourimonster-vyla-api.hf.space/api/test/1396?season=1&episode=1&source=source2"
```

```javascript JavaScript
// Test a single provider
const res = await fetch(
  'https://missourimonster-vyla-api.hf.space/api/test/550?source=source1'
);
const data = await res.json();

if (data.ok) {
  console.log(`✓ ${data.source} (${data.elapsed_ms}ms): ${data.url}`);
} else {
  console.error(`✗ ${data.source}: ${data.error}`);
}
```

```typescript TypeScript
interface TestResponse {
  source: string;
  id: string;
  s: number | null;
  e: number | null;
  ok: boolean;
  url: string | null;
  raw_url: string | null;
  elapsed_ms: number;
  error: string | null;
}

const data: TestResponse = await fetch(
  'https://missourimonster-vyla-api.hf.space/api/test/550?source=source1'
).then(r => r.json());
```

```python Python
import requests

res = requests.get(
  'https://missourimonster-vyla-api.hf.space/api/test/550',
```

## Response

<ResponseField name="source" type="string" required>
  The provider key that was tested.
</ResponseField>

<ResponseField name="id" type="string" required>
  The TMDB ID that was tested.
</ResponseField>

<ResponseField name="s" type="number | null" required>
  Season number. `null` for movies.
</ResponseField>

<ResponseField name="e" type="number | null" required>
  Episode number. `null` for movies.
</ResponseField>

<ResponseField name="ok" type="boolean" required>
  `true` if the provider returned a non-null URL. Does **not** guarantee the URL is playable — use `/api/movie` or `/api/tv` for verified sources.
</ResponseField>

<ResponseField name="url" type="string | null">
  Proxied stream URL if the provider returned one. `null` on failure.
</ResponseField>

<ResponseField name="raw_url" type="string | null">
  The original, unproxied stream URL from the provider. Useful for debugging. `null` on failure.
</ResponseField>

<ResponseField name="elapsed_ms" type="number" required>
  Time in milliseconds from the start of the test to receiving a URL (or failure).
</ResponseField>

<ResponseField name="error" type="string | null" required>
  Error message if the provider threw an exception or returned nothing. `null` on success.
</ResponseField>

## Testing All Sources

To test all providers in a loop and find which are working for a given title:

```javascript
const BASE    = 'https://missourimonster-vyla-api.hf.space';
const SOURCES = ['source1', 'source2', 'source3', 'source4', 'source5', 'source6'];

async function testAll(tmdbId, season, episode) {
  const params = season && episode
    ? `?season=${season}&episode=${episode}`
    : '';

  const results = await Promise.all(
    SOURCES.map(source =>
      fetch(`${BASE}/api/test/${tmdbId}${params}&source=${source}`)
        .then(r => r.json())
        .catch(err => ({ source, ok: false, error: err.message, elapsed_ms: null }))
    )
  );

  results.forEach(r => {
    const status = r.ok ? '✓' : '✗';
    const time   = r.elapsed_ms != null ? `${r.elapsed_ms}ms` : 'N/A';
    console.log(`${status} ${r.source.padEnd(8)} ${time.padStart(7)}  ${r.error || ''}`);
  });
}

testAll(550);
// ✓ source1   1204ms
// ✓ source2    980ms
// ✗ source3     435ms  no stream returned
// ✓ source4   1540ms
// ✓ source5   1100ms
// ✓ source6    620ms
```

<Tip>
  For a quicker overview, use `/api/health` — it probes all providers simultaneously and is optimised for monitoring.
</Tip>
