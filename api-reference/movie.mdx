---
title: Movie Sources
api: "GET https://missourimonster-vyla.hf.space/api/movie"
description: Fetch all working HLS sources, subtitle tracks, and TMDB metadata for a movie.
---

Queries all configured providers in parallel, verifies each candidate URL live, and returns only the streams that are actually playable. Subtitles and TMDB metadata are fetched concurrently and bundled into the same response.

Every `url` in the response is already routed through the HLS proxy with M3U8 segment paths rewritten — pass it directly to `hls.loadSource()` after prepending the base URL.

---

## Query Parameters

<ParamField query="id" type="string" required>
  TMDB movie ID. Find it on [themoviedb.org](https://www.themoviedb.org) — it's the number in the URL.

  **Example:** `themoviedb.org/movie/550` → `id=550`
</ParamField>

## Request


```javascript JavaScript
const res  = await fetch('https://missourimonster-vyla.hf.space/api/movie?id=550');
```

```python Python
import requests

data = requests.get(
  'https://missourimonster-vyla.hf.space/api/movie',
  params={'id': 550}
).json()

BASE = 'https://missourimonster-vyla.hf.space'
for source in data['sources']:
    print(f"{source['label']}: {BASE}{source['url']}")
```

```go Go
package main

import (
  "encoding/json"
  "fmt"
  "net/http"
)

type Source struct {
  Source string `json:"source"`
  Label  string `json:"label"`
  URL    string `json:"url"`
}

type MovieResponse struct {
  Sources   []Source               `json:"sources"`
  Subtitles []map[string]string    `json:"subtitles"`
  Meta      map[string]interface{} `json:"meta"`
}

func main() {
  res, _ := http.Get("https://missourimonster-vyla.hf.space/api/movie?id=550")
  defer res.Body.Close()

  var data MovieResponse
  json.NewDecoder(res.Body).Decode(&data)

  base := "https://missourimonster-vyla.hf.space"
  for _, s := range data.Sources {
    fmt.Printf("%s: %s%s\n", s.Label, base, s.URL)
  }
}
```

## Response

<ResponseField name="sources" type="Source[]" required>
  Array of verified, proxied HLS sources. Only streams that passed the live `#EXTM3U` verification check are included.

  <Expandable title="Source object">
    <ResponseField name="source" type="string">
      Internal provider key. Matches the keys in `/api/health` and `/api/test`.
    </ResponseField>
    <ResponseField name="label" type="string">
      Human-readable provider name.
    </ResponseField>
    <ResponseField name="url" type="string">
      Proxied HLS stream URL. All M3U8 segment and encryption key URIs are rewritten to route through the proxy. **Prepend `https://missourimonster-vyla.hf.space`** before passing to `hls.loadSource()`.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="subtitles" type="Subtitle[]" required>
  Available subtitle tracks. Empty array `[]` if none are found for this title.

  <Expandable title="Subtitle object">
    <ResponseField name="label" type="string">
      Language name, e.g. `English`, `Spanish`, `Arabic`, `French`.
    </ResponseField>
    <ResponseField name="file" type="string">
      Direct URL to a subtitle file (`.vtt` or `.srt`). Use as `<track src="...">` or pass to your player's subtitle API.
    </ResponseField>
    <ResponseField name="type" type="string">
      Subtitle format — `vtt` or `srt`.
    </ResponseField>
    <ResponseField name="source" type="string">
      Internal subtitle source identifier.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="meta" type="object | null">
  TMDB movie metadata. `null` if no `TMDB_API_KEY` is configured on the server. When present, this is the raw TMDB API response for the movie.
</ResponseField>

## Notes

<AccordionGroup>
  <Accordion title="Response time">
    Typical response time is **3–8 seconds**. The request waits for all providers to respond or time out — the slowest active provider determines the ceiling. Always show a loading state in your UI.
  </Accordion>
  <Accordion title="Result ordering">
    Sources are returned in the order they passed verification, which generally correlates with provider speed. The first source is usually the fastest-responding one.
  </Accordion>
  <Accordion title="Empty sources vs 502">
    A `502` with `"no working sources found"` means all providers returned nothing. A `200` with an empty `sources` array should not occur — this is handled server-side.
  </Accordion>
  <Accordion title="Using the URL field">
    The `url` field is a relative path starting with `/api?url=...`. Build an absolute URL like this:

    ```javascript
    const streamUrl = 'https://missourimonster-vyla.hf.space' + source.url;
    hls.loadSource(streamUrl);
    ```
  </Accordion>
</AccordionGroup>
