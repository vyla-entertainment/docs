---
title: Movie Sources
api: "GET https://missourimonster-vyla-api.hf.space/api/movie"
description: Fetch all working HLS sources, subtitle tracks, and TMDB metadata for a movie.
---

Queries all six providers in parallel, verifies each candidate URL live, and returns only the streams that are actually playable. Subtitles and TMDB metadata are fetched concurrently and bundled into the same response.

Every `url` in the response is already routed through the Cloudflare HLS proxy with M3U8 segment paths rewritten — pass it directly to `hls.loadSource()`.

---

## Query Parameters

<ParamField query="id" type="string" required>
  TMDB movie ID. Find it on [themoviedb.org](https://www.themoviedb.org) — it's the number in the URL.

  **Example:** `themoviedb.org/movie/550` → `id=550`
</ParamField>

---

## Request

<CodeGroup>
```bash cURL
curl "https://missourimonster-vyla-api.hf.space/api/movie?id=550"
```

```javascript JavaScript
const res  = await fetch('https://missourimonster-vyla-api.hf.space/api/movie?id=550');
const data = await res.json();
```

```go Go
package main

import (
  "encoding/json"
  "fmt"
  "net/http"
)

type Source struct {
  Source string `json:"source"`
  Label  string `json:"label"`
  URL    string `json:"url"`
}

type MovieResponse struct {
  Sources   []Source               `json:"sources"`
  Subtitles []map[string]string    `json:"subtitles"`
  Meta      map[string]interface{} `json:"meta"`
}

func main() {
  res, _ := http.Get("https://missourimonster-vyla-api.hf.space/api/movie?id=550")
  defer res.Body.Close()

  var data MovieResponse
  json.NewDecoder(res.Body).Decode(&data)

  for _, s := range data.Sources {
    fmt.Printf("%s: https://missourimonster-vyla-api.hf.space%s\n", s.Label, s.URL)
  }
}
```

## Response

<ResponseField name="sources" type="Source[]" required>
  Array of verified, proxied HLS sources. Only streams that passed the live `#EXTM3U` verification check are included. Array may be empty if all providers fail.

  <Expandable title="Source object">
    <ResponseField name="source" type="string">
      Internal provider key, e.g. `source1`, `source2`. Matches the keys in the [providers table](/introduction#providers).
    </ResponseField>
    <ResponseField name="label" type="string">
      Human-readable provider name, e.g. `Source1`, `Source2`.
    </ResponseField>
    <ResponseField name="url" type="string">
      Proxied HLS stream URL. All M3U8 segment and encryption key URIs are rewritten to route through the same Cloudflare origin. Pass directly to `hls.loadSource()`.

      **Important:** Prepend `https://missourimonster-vyla-api.hf.space` if your player requires an absolute URL.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="subtitles" type="Subtitle[]" required>
  Available subtitle tracks. Empty array `[]` if none are found for this title.

  <Expandable title="Subtitle object">
    <ResponseField name="label" type="string">
      Language name, e.g. `English`, `Spanish`, `Arabic`, `French`.
    </ResponseField>
    <ResponseField name="file" type="string">
      Direct URL to a WebVTT (`.vtt`) file. Use as `<track src="...">` or pass to your player's subtitle API.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="meta" type="object | null">
  TMDB movie metadata. `null` if no `TMDB_API_KEY` is configured on the server. When present, this is the raw TMDB API response for the movie.
</ResponseField>

## Notes

<AccordionGroup>
  <Accordion title="Response time">
    Typical response time is **3–8 seconds**. The request completes once all providers have responded (or timed out) and Source1 has a 15s timeout; Source3 can take up to 25s — these are the usual bottlenecks on slow responses.
  </Accordion>
  <Accordion title="Result ordering">
    Sources are returned in the order they passed verification, which correlates with provider speed. The first source is generally the fastest-responding provider.
  </Accordion>
  <Accordion title="Empty sources array vs 502">
    A `502` with `"no working sources found"` means all providers returned nothing. A `200` with an empty `sources` array should not occur — this is handled server-side.
  </Accordion>
  <Accordion title="Using the URL field">
    The `url` field is a relative path beginning with `/api?url=...`. Prepend the base URL when constructing absolute URLs for your player:

    ```javascript
    const streamUrl = 'https://missourimonster-vyla-api.hf.space' + source.url;
    hls.loadSource(streamUrl);
    ```
  </Accordion>
</AccordionGroup>
