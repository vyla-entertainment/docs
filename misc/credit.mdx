This page highlights the open-source projects and developers that heavily inspired the Vyla API, player ecosystem, and overall UI direction.

---

## Player & UI Inspiration

The overall player experience, streaming layout, and much of the UI direction for Vyla was heavily inspired by P-Stream.

- Repository: https://github.com/p-stream
- Owner: https://github.com/Pasithea0

Their work served as a major reference point for the design language and viewing experience used throughout Vyla.

---

## Stream API Inspiration

The architecture and implementation ideas behind the Vyla streaming API were heavily inspired by CinePro.

### Resources

- Organization: https://github.com/cinepro-org/
- Main Website: https://cinepro.cc/
- Documentation: https://docs.cinepro.cc/introduction
- UI Components: https://ui.cinepro.cc/
- Owner: https://github.com/An0n-00

CinePro's ecosystem played a large role in shaping how Vyla handles providers, source structures, streaming workflows, and API organization.

---

## Source Implementation & Fixes

The owner of Movieslay helped fix and implement new sources across the Vyla API.

- Repository: https://github.com/klashdevelopment/MovieslayEmerald
- Discord: https://discord.gg/rD75qxGBVG
- Owner: https://github.com/gavingogaming

Movieslay Emerald is a successor to the original Movieslay webapp. gavingogaming's direct contributions were instrumental in expanding and stabilizing Vyla's source coverage.
