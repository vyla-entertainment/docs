<Tip>

</Tip>

<iframe
  src="https:///?id=8363"
  allow="fullscreen"
  style={{ width: '100%', aspectRatio: '16/9', border: 'none', borderRadius: '12px' }}
/>

<Tip>
  The player defaults to TMDB ID `8363` (Superbad). To load a different title, change the `id` in the URL above. For TV, use `?type=tv&id=<id>&season=1&episode=1`.
</Tip>

---

## URL Format

**Movie**

```
https:///?id=<tmdb_id>
```

**TV Show**

```
https:///?type=tv&id=<tmdb_id>&season=<season>&episode=<episode>
```

---

## Parameters

| Parameter | Description |
|-----------|-------------|
| `type` | `movie` (default) or `tv` |
| `id` | TMDB ID of the title |
| `season` | Season number — TV only |
| `episode` | Episode number — TV only |

---

## Embed It Yourself

Copy this into any HTML page to embed the player:

### Movies

```html
<iframe
  src="https:///?id=8363"
  allowfullscreen
  style="width:100%;aspect-ratio:16/9;border:none;border-radius:12px"
></iframe>
```

### TV Shows

```html
<iframe
  src="https:///?type=tv&id=1399&season=1&episode=1"
  allowfullscreen
  style="width:100%;aspect-ratio:16/9;border:none;border-radius:12px"
></iframe>
```

<Note>
  The player pulls streams from the same providers as the API. If a title shows no sources, check [`/api/health`](https://missourimonster-vyla.hf.space/api/health) to confirm providers are live.
</Note>
